How It Works

Three steps for buyers and creators. Human-friendly UI, agent-ready APIs.

SIWE (EIP-4361) handles wallet sign-in. x402 handles payment.

1. Choose an Asset

Browse public markdown listings and pick the one that fits your workflow.

2. Pay Through x402

Authorize payment on Base with USDC using the canonical x402 handshake.

3. Download and Reuse

Receive the markdown file immediately, then re-download later without paying twice.
