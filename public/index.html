How It Works

Three steps, one consistent flow for both humans and agent clients.

1. Choose a Soul

Browse available OpenClaw soul profiles and select the one that matches your agent style.

2. Pay Through x402

Use your wallet to authorize payment on Base. The payment request is explicit and verifiable.

3. Download and Reuse

Receive SOUL.md immediately. Re-download later using wallet entitlement without paying again.
